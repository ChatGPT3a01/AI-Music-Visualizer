# 📖 Eye Music 使用指南

## 快速開始

### 第一次使用

1. **安裝環境**
   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt

   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

2. **安裝 FFmpeg**
   - Windows: 下載並加入 PATH
   - macOS: `brew install ffmpeg`
   - Linux: `sudo apt install ffmpeg`

3. **啟動應用**
   ```bash
   # Windows
   start_windows.bat

   # macOS/Linux
   ./start.sh
   ```

4. **開啟瀏覽器**
   - 訪問 `http://localhost:5000`

## 詳細操作步驟

### 步驟 1: 上傳音樂

1. 點擊上傳區域或拖放檔案
2. 支援的格式：
   - MP3
   - WAV
   - FLAC
   - OGG
   - M4A
   - AAC
3. 檔案上傳後會自動分析音頻資訊

### 步驟 2: 設定參數

#### 快速選擇模板

系統提供 4 個預設模板：

1. **預設** - 標準頻譜能量條，彩虹色
2. **圓形頻譜** - 放射狀效果，紫色系
3. **鋼琴瀑布流** - 音符瀑布，海洋色
4. **火焰效果** - 能量條，火焰色

#### 自訂參數

**視覺化類型：**
- 頻譜能量條：經典的垂直能量條
- 圓形頻譜：圓形放射狀
- 鋼琴瀑布流：音符瀑布效果

**色彩方案：**
- 彩虹色：多彩漸變
- 火焰色：紅橙黃漸變
- 海洋色：藍綠色系
- 紫色系：紫色漸變

**影片解析度：**
- 1280x720 (HD)
- 1920x1080 (Full HD) ⭐ 推薦
- 2560x1440 (2K)
- 3840x2160 (4K)

**影片幀率 (FPS)：**
- 24 FPS：電影風格
- 30 FPS：標準 ⭐ 推薦
- 60 FPS：流暢效果（檔案較大）

**頻譜高度倍率 (0.5 - 2.0)：**
- 控制頻譜的高度
- 建議值：1.0 - 1.5

**頻譜透明度 (0.1 - 1.0)：**
- 控制視覺效果的透明度
- 建議值：0.7 - 0.9

**背景顏色：**
- 使用顏色選擇器自訂背景色
- 黑色背景效果最佳

### 步驟 3: 生成影片

1. 點擊「開始生成」
2. 等待處理（視檔案大小和設定而定）
3. 進度條會顯示即時進度
4. 完成後點擊「下載影片」

## 進階技巧

### 最佳設定建議

**流行音樂：**
- 視覺化類型：頻譜能量條
- 色彩方案：彩虹色
- 高度倍率：1.2
- 透明度：0.8

**電子音樂：**
- 視覺化類型：圓形頻譜
- 色彩方案：火焰色
- 高度倍率：1.5
- 透明度：0.9

**古典音樂：**
- 視覺化類型：鋼琴瀑布流
- 色彩方案：海洋色
- 高度倍率：1.0
- 透明度：0.7

### 效能優化

**加快生成速度：**
1. 降低解析度（使用 720p）
2. 降低幀率（使用 24-30 FPS）
3. 縮短音頻長度

**提升影片品質：**
1. 使用高解析度（1080p 或 2K）
2. 使用 30-60 FPS
3. 調整頻譜參數以獲得最佳視覺效果

### 常見問題

**Q: 生成需要多長時間？**
A: 取決於多個因素：
- 音頻長度
- 影片解析度
- 幀率設定
- 電腦效能

一般來說，3 分鐘的音頻使用 1080p/30FPS 需要約 5-10 分鐘。

**Q: 為什麼影片檔案很大？**
A: 可能原因：
- 解析度過高
- 幀率過高
- 可以使用影片壓縮工具進一步壓縮

**Q: 生成失敗怎麼辦？**
A: 檢查項目：
1. FFmpeg 是否正確安裝
2. 磁碟空間是否足夠
3. 音頻檔案是否損壞
4. 查看終端的錯誤訊息

**Q: 可以批量處理嗎？**
A: 目前版本不支援批量處理，需要逐個處理。

**Q: 生成的影片可以商用嗎？**
A: 可以，但請遵守音樂版權相關法律。

## 系統需求

### 最低需求
- CPU: 雙核心處理器
- RAM: 4GB
- 磁碟空間: 5GB
- Python: 3.8+

### 推薦配置
- CPU: 四核心以上處理器
- RAM: 8GB+
- 磁碟空間: 10GB+
- Python: 3.10+
- 獨立顯示卡（加速渲染）

## 檔案輸出

**輸出位置：**
`src/static/outputs/`

**檔案格式：**
- 影片格式：MP4 (H.264)
- 音頻編碼：AAC
- 可用任何影片播放器播放
