  • 步驟三：在 AI 小幫手後台完成設定
  • 最終目標：成功在 LINE 上與你的專屬 AI 助理對話！

開始前的準備工作

在我們開始之前，請確認你已經擁有以下兩個帳號。整個過程完全免費。

1. 你的個人 LINE 帳號

這將用來登入 LINE 的官方後台，以建立及管理你的機器人。

2. AI 小幫手 (thisnote.space) 帳號

這是我們 AI 機器人的大腦。你需要註冊一個帳號來取得串接服務。

步驟一：建立 LINE 官方帳號

首先，我們要為你的 AI 助理建立一個在 LINE 上的「家」。

  1. 前往 LINE Official Account Manager 網站。
  2. 點擊「免費開設帳號」，並使用你的個人 LINE 帳號登入。
  3. 依照畫面指示，填寫你的「帳號名稱」（這會是你的機器人名稱）、電子郵件與公司/品牌資訊。
  4. 完成建立後，你會得到一個官方帳號，並進入管理後台。

提示：

帳號名稱可以取個有趣的名字，例如「小亮的 AI大腦」或「萬事通助理」。這個名稱之後還可以修改。

步驟二：啟用 Messaging API

為了讓「AI 小幫手」平台能控制你的 LINE 帳號，我們需要啟用 API (應用程式介面) 功能。

  1. 在 LINE 官方帳號後台，點擊右上角的「設定」。
  2. 在左側選單中，點選「Messaging API」。
  3. 畫面會顯示「啟用 Messaging API」，勇敢地按下去！
  4. 接著，你需要建立一個「服務提供者 (Provider)」，可以填寫你自己的名字或品牌名稱，然後按「確定」。
  5. 完成後，你就成功開啟了讓外部服務與你的 LINE 帳號溝通的橋樑！

核心概念：

啟用 Messaging API 就像是把機器人的「遙控權」交出來，讓 `thisnote.space` 平台可以代替你回覆訊息。

步驟三：取得 LINE API 金鑰

接下來是最關鍵的一步：我們要從 LINE Developers 取得兩組重要的「金鑰」，它們就像是 AI 助理的帳號密碼。

在啟用 Messaging API 之後，你會被導向到 LINE Developers 的設定頁面。請找到並複製以下兩項資訊：

1. Channel Secret (通道祕鑰)

Channel Secret

這是一長串的字元，代表你的機器人身份。請直接複製它。

2. Channel Access Token (通道存取權杖)

Channel access token (long-lived)

你需要點擊「Issue (發行)」按鈕來產生一組新的權杖，然後再複製它。

請妥善保管這兩組金鑰，不要洩漏給他人！我們下一步就會用到它們。

步驟四：設定 AI 小幫手後台

現在，帶著我們剛剛取得的兩把金鑰，回到「AI 小幫手」的家，告訴它如何與你的 LINE 帳號聯繫。

  1. 登入 thisnote.space 的 Bot Manager。
  2. 新增一個機器人，並為它取一個好記的名字。
  3. 你會看到兩個欄位：
    • Channel Secret
    • Channel Access Token
  4. 將上一步從 LINE Developers 複製的兩組金鑰，分別貼到對應的欄位中。
  5. 儲存設定。此時，AI 小幫手的後台就已經知道要跟哪個 LINE 帳號溝通了。

步驟五：設定 Webhook，完成最後一哩路！

我們已經建立了從 AI 小幫手到 LINE 的單向連結。現在要設定反向的路，讓 LINE 知道要把收到的訊息傳給誰處理。

  1. 在 AI 小幫手 (thisnote.space) 的機器人設定頁面，複製「Webhook URL」。
    Webhook URL
  2. 回到 LINE Developers 的設定頁面。
  3. 找到「Webhook settings」區塊，將剛剛複製的 URL 貼到「Webhook URL」欄位中，並點擊「Update」。
  4. 最重要的一步：將下方的「Use webhook」開關切換為啟用 (Enabled) 狀態。

🎉 恭喜！雙向串接已完成！

現在起，任何傳送到你 LINE 官方帳號的訊息，都會被自動轉發到 AI 小幫手平台進行處理。

步驟六：測試你的 AI 助理！

是時候跟你的新助理打聲招呼了！

  1. 回到 LINE 官方帳號後台，在「帳號設定」中可以找到你機器人的「基本ID」或「QR Code」。
  2. 用你的手機 LINE App，透過 ID 搜尋或掃描 QR Code，將你的機器人加為好友。
  3. 打開聊天室，隨意傳送一則訊息，例如「你好，你是誰？」或「幫我規劃明天去台北的行程」。
  4. 如果設定正確，你應該會立刻收到來自 AI 小幫手的回覆！

疑難排解：

如果沒有收到回覆，請回頭檢查步驟 5 和 7 的金鑰與 Webhook URL 是否都已正確複製貼上，且 Webhook 功能已啟用。

你的 AI 助理能做什麼？

這個 AI 助理不只會聊天，它還內建了許多強大功能，能幫你處理各種任務：

會議/語音轉文字

在對話框輸入「會議錄音模式」，接著傳送語音訊息，AI 就會自動將語音轉為帶有標點符號的逐字稿。

文件分析與問答

在 AI 小幫手後台上傳 PDF 或其他文件，你就可以在 LINE 上直接對文件內容進行提問，讓 AI 幫你找答案、做總結。

AI 繪圖

輸入「畫圖」加上你的描述 (例如：「畫圖 一隻貓在太空中」)，AI 就會為你生成獨一無二的圖片。
