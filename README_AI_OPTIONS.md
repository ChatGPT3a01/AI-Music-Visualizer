## 💡 使用建議

### 情境 1：個人學習 / 測試專案

**推薦方案**：只用 Gemini Flash ✅

**理由**：
- ✅ 完全免費
- ✅ 速度夠快
- ✅ 品質足夠日常使用
- ✅ 每天 60 次額度很夠用

**成本**：$0 / 月

---

### 情境 2：小型商業專案（< 100 任務/月）

**推薦方案**：Gemini Flash + Pro 混用

**理由**：
- ✅ 快速任務用 Flash（免費）
- ✅ 重要任務用 Pro（品質高）
- ✅ 成本可控

**預估成本**：$0-10 / 月

---

### 情境 3：中型商業專案（100-1000 任務/月）

**推薦方案**：Gemini Flash（分析） + GPT-5.6 Terra（描述）

**理由**：
- ✅ 分析用 Gemini（省錢）
- ✅ 描述用 GPT-5.6 Terra（品質好）
- ✅ 成本與品質平衡

**預估成本**：$20-50 / 月

---

### 情境 4：大型商業專案（> 1000 任務/月）

**推薦方案**：三者混用 + 智能路由

**策略**：
```
普通任務（80%） → Gemini Flash（免費）
重要任務（15%） → Gemini Pro（中等）
頂級任務（5%）  → GPT-5.6 Terra（最佳品質）
```

**理由**：
- ✅ 成本優化（大部分用免費）
- ✅ 品質保證（關鍵任務用 GPT）
- ✅ 備援機制（一個失敗自動切換）

**預估成本**：$100-300 / 月

## 🔄 自動備援機制

系統已內建智能路由，會自動處理 API 失敗情況：

```python
# 在 ai_service.py 中
def smart_analyze(song_name):
    # 優先使用 Gemini（速度快、成本低）
    if gemini_available:
        return analyze_with_gemini()

    # Gemini 不可用時，自動切換到 GPT
    elif openai_available:
        return analyze_with_gpt()

    # 都不可用時，使用預設參數
    else:
        return default_parameters()
```

**備援順序**：
1. Gemini Flash（優先）
2. GPT-5.6 Terra（備援）
3. 預設參數（最後保障）

## ❓ 常見問題

### Q1: 我只有 Gemini，能正常使用所有功能嗎？

**A**: 可以！系統會自動使用 Gemini 處理所有任務：
- 音樂分析 ✅
- 參數推薦 ✅
- 描述生成 ✅（使用 Gemini Pro）
- 社群文案 ✅（使用 Gemini Pro）

### Q2: 我只有 OpenAI，能正常使用嗎？

**A**: 可以！系統會使用 GPT 處理所有任務：
- 音樂分析 ✅（GPT-5.6 Terra）
- 參數推薦 ✅（GPT-5.6 Terra）
- 描述生成 ✅（GPT-5.6 Terra）
- 社群文案 ✅（GPT-5.6 Terra）

### Q3: 兩個都有會怎麼分配工作？

**A**: 系統預設策略（可自訂）：
- 音樂分析 → Gemini Flash（快速且免費）
- 深度分析 → Gemini Pro（深度思考）
- 描述生成 → GPT-5.6 Terra（文字品質高）
- 社群文案 → GPT-5.6 Terra（創意性強）

### Q4: 可以自訂使用哪個模型嗎？

**A**: 可以！在 Google Sheets 的「AI模型」欄位選擇：
- Gemini 3.5 Flash
- Gemini 3.1 Pro
- GPT-5.6 Terra
- GPT-5.6 Terra

### Q5: 如果 API 失敗會怎樣？

**A**: 系統有三層保護：
1. 重試機制（自動重試 3 次）
2. 備援切換（切換到另一個 AI）
3. 預設參數（最後保障，不會完全失敗）
