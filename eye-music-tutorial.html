開始前的準備工作

在開始建置之前，請確認你的系統已安裝以下環境：

1. Python 3.8+

主要開發語言，用於後端和音頻處理

python --version # 檢查版本
# 應顯示 Python 3.8.0 或更高版本

2. FFmpeg

影片處理工具，必須安裝

# Windows: 下載並加入 PATH
# macOS: brew install ffmpeg
# Linux: sudo apt install ffmpeg

3. 文字編輯器

推薦使用 VS Code、PyCharm 或任何你喜歡的編輯器

4. 系統資源

建議至少 4GB RAM 和 5GB 可用磁碟空間

步驟 1：建立專案結構

首先建立專案目錄和基本結構：

mkdir Eye_Music
cd Eye_Music
mkdir -p src/templates src/static/{css,js,uploads,outputs} src/core

📁 完整目錄結構：

Eye_Music/
├── src/
│ ├── app.py # Flask 主程式
│ ├── core/
│ │ ├── __init__.py
│ │ └── visualizer.py # 視覺化核心引擎
│ ├── templates/
│ │ └── index.html # 前端頁面
│ └── static/
│ ├── css/
│ │ └── style.css # 樣式表
│ ├── js/
│ │ └── app.js # 前端邏輯
│ ├── uploads/ # 上傳音頻
│ └── outputs/ # 輸出影片
├── requirements.txt # 依賴套件
├── start_windows.bat # Windows 啟動腳本
└── README.md # 說明文件

提示：

這個結構採用 MVC 模式，將核心邏輯、視圖和控制器分離，便於維護和擴展。

步驟 2：安裝依賴套件

建立 requirements.txt 並安裝所需套件：

# requirements.txt 內容
Flask==3.0.0
flask-socketio==5.3.5
python-socketio==5.10.0
eventlet==0.33.3
librosa==0.10.1
moviepy==1.0.3
numpy==1.24.3
scipy==1.11.4
Pillow==10.1.0

建立虛擬環境（推薦）：

# Windows
python -m venv venv
venv\Scripts\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate

安裝套件：

pip install -r requirements.txt

安裝時間約 5-10 分鐘，請耐心等候

步驟 5：設計前端界面

建立現代化的 Web 界面，包含三個主要部分：

HTML 結構 (templates/index.html)

  • • 步驟指示器（1.上傳 → 2.設定 → 3.生成）
  • • 拖放上傳區域
  • • 參數設定面板（預設模板、色彩、解析度等）
  • • 進度顯示與結果下載

CSS 樣式 (static/css/style.css)

  • • 深色主題設計
  • • 漸變背景與卡片效果
  • • 響應式佈局（支援手機/平板/桌面）
  • • 動畫過渡效果

JavaScript 邏輯 (static/js/app.js)

  • • Socket.IO 連接與即時通訊
  • • 拖放檔案上傳
  • • AJAX API 呼叫
  • • 進度條更新
  • • Toast 通知系統

步驟 6：啟動與測試

現在可以啟動應用程式並進行測試了！

啟動方式：

Windows 使用者：

雙擊 start_windows.bat

macOS/Linux 使用者：

chmod +x start.sh
./start.sh

手動啟動：

cd src
python app.py

開啟瀏覽器

訪問：

http://localhost:5000

測試流程：

  1. 上傳一首音樂檔案（建議先用短音樂測試，約 30-60 秒）
  2. 選擇喜歡的視覺化類型和色彩方案
  3. 調整參數（建議先用預設值）
  4. 點擊「開始生成」
  5. 觀察進度條和即時訊息
  6. 生成完成後下載影片
