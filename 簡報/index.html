STEP 1下載專案 & 建立虛擬環境

# 下載專案 git clone https://github.com/ChatGPT3a01/AI-Music-Visualizer.git cd AI-Music-Visualizer # 建立並啟用虛擬環境（Windows） python -m venv venv venv\Scripts\activate # Mac / Linux 改用： python3 -m venv venv && source venv/bin/activate
💡 虛擬環境 = 幫這個專案開一個「獨立的套件房間」，不會跟其他專案打架。啟用後提示字前面會出現 (venv)。

STEP 2安裝套件 & FFmpeg

# 安裝 Python 套件（Flask、Socket.IO、librosa…） pip install -r requirements.txt

再裝 FFmpeg（負責把畫面壓成影片）：

  • Mac：brew install ffmpeg
  • Linux：sudo apt install ffmpeg
  • Windows：到 ffmpeg.org/download 下載 → 解壓到 C:\ffmpeg → 把 C:\ffmpeg\bin 加進系統 PATH。
⚠️ 沒裝 FFmpeg，分析可以跑但「輸出影片」會失敗。裝完重開終端機讓 PATH 生效。

STEP 3啟動平台

python app.py * Running on http://localhost:5000 ← 啟動成功

打開瀏覽器前往 http://localhost:5000，會看到 Eye Music 的上傳介面。

✅ 沒有 AI API Key 也能用！核心的「上傳→分析→生成影片」照跑，只是少了 AI 推薦參數。要 AI 推薦再把金鑰填進 .env。

STEP 4生成你的第一支視覺化影片

  1. 拖放上傳一首音樂檔（mp3／wav）。
  2. 選視覺化類型（先試「頻譜能量條」）與色彩方案（先試「彩虹色」）。
  3. 調解析度／幀率（新手用預設即可）。
  4. 按「生成」，看即時進度條（Socket.IO 回報）。
  5. 完成後預覽 / 下載 MP4。
🎉 你已經做出一台「音樂轉影片」的機器了！
